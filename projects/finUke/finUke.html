findUke


This is a full-stack website and authorizes admin, seller, and buyer. Buyers can book their resale product and payments using a card.A seller can add their product and Admin can handle buyers and sellers delete them and review them.
Its it created with mainly:- 1.React JS. 2. Tailwind Css. 3.Daisy Ui 4.Firebase 5.Vercel 6.React Query 7.React Hot Toast etc.
Main Features--

There is a home page about the site and the product that have resale offer.
Click on the product, user will be redirect to login as well as sign up.
Sign up page has two types of user as buyer and seller.
User will see select a category of product and the items that are available.
By booking , user will book the product and as per user is buyer or seller, dashboard will render.
Admin can log in and see all buyer and seller and can also choose to make admin.
It is Authenticated with Firebase. User can log in with email,password, google sign in, basically.
Footer is there to have clear idea about the authority.
Contact page is to contact with authority.
There is a blog section to answer the asked question. About page is about answering the asked question. Contact page is given to contact with the admin. Responsive SPA site etc...
